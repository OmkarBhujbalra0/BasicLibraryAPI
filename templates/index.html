What is BasicLibAPI?

BasicLibAPI is an REST(Representational State Transfer) API with the help of which you can learn about basic CRUD (Create , Read , Update , Delete) operations. Using BasicLibAPI you will learn how to use HTTP Requests ( GET,PUT,POST,DELETE ). Learn operating elements on API like managing books in Library.
